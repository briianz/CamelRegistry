# Camel Registry - Teve Nyilvántartó
Ez a projekt egy két részből álló próba feladat.

Technológiai rész:

Backend: .NET 8 Minimal API 


Adatbázis: SQLite + Entity Framework Core 


Dokumentáció: OpenAPI / Swagger UI 


Tesztelés: xUnit 


Frontend (Fejlesztés alatt): Angular 17+, Bootstrap 

Funkciók (Backend):

CRUD műveletek: Tevék létrehozása, listázása, módosítása és törlése .


Validáció: Szigorú púp-szám ellenőrzés (1 vagy 2 púp).


Automatizáció: Az SQLite adatbázis fájl automatikus létrehozása indításkor.


Swagger: Interaktív tesztelő felület a /swagger útvonalon.

Telepítés és Futtatás:
Klónozd a repository-t.

Lépj be a CamelRegistry mappába.

Futtasd az alkalmazást: dotnet run

Nyisd meg a böngészőben a Swagger felületet.
